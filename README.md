# RPlayer
拓展性丰富、小巧的HTML5播放器，可以方便地嵌入您的Web项目中

## 界面

![](docs/main.png)

## 特性
* 外链播放
* 渐变色主题切换
* 丰富的快捷键操作
    * S：播放/ 暂停
## 兼容性
* 浏览器支持
  * Internet Explorer 9+
  * Chrome/Firefox/Microsoft Edge/Opera/Safari
* 音频格式支持
  
||Chrome|Firefox|IE9|Opera|Safari|
